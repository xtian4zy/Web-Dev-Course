CSS INTRODUCTION

CSS stands for Cascading Stylesheet and is used to describe how HTML elements should be displayed. CSS is used to add beauty to our web page so that it can be visually appealing to users. We can add CSS to our web page using either the Inline, Internal, or External methods.

"If you want an asthethic page, use CSS to style it."
